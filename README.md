## 使用介绍&注意事项
### 方法一：
下载使用lookcos动态桌面便携版，直接运行exe即可。

http://look.hackhx.com/EXE/LookCos_DJS.zip

优点：便携快速，仅5M，不适合老年机使用。

缺点：杀毒软件误报。

需要退出倒计时只需要 右键右下角托盘无标，点击退出，即可

### 一些说明：

程序无毒无副作用，仅仅用户添加开机启动时会申请管理员权限。
关于报毒：程序加了个壳故而有的杀毒软件会报毒，已经提交360、电脑管家处理！
